OBJECT SENSING SMART DUSTBIN
The Object Sensing Smart Dustbin was a project which was made by us as a college project. It used the microcontroller Arduino Uno and a Wi-Fi module. It senses the waste material placed near it using a IR Sensor and it uses a moisture sensor to segregate between dry and wet waste. When the dustbin is full, the owner receives an email regarding the same.
